HEXO

发表于 2018-04-08

Hexo是一个免费开源的博客框架。

目前个人博客主要分为两种，一种是在CSDN、博客园、Segment Fault等网站开通个人博客；另一种就是使用开源的博客框架，如：WordExpress，Hexo，Jekyll，搭建自己的博客。第一种很不“程序员”^_^，所以直接否决。说来惭愧，我选择Hexo而不是其他博客框架，并不是基于性能等方面的考虑，而是觉得Hexo的界面最符合我的审美观，zzz。

Hexo框架和WordPress比起来算不上成熟，但是使用的人数其实还是挺多的。网上的相关教程也比较详细了，这里更多的是对其他教程总结——方便之后的查阅。

主题配置

关于配置文件：

在 Hexo 中有两份主要的配置文件，其名称都是 _config.yml。 其中，一份位于站点根目录下，主要包含 Hexo 本身的配置，称为“站点配置文件”；另一份位于主题目录下，这份配置由主题作者提供，主要用于配置主题相关的选项，称为“主题配置文件”。

主要需要配置的项目包括：

  • 个人资料
  • Scheme。NexT最强大的一点是，他提供了三个Scheme。每个Scheme都有不同的外观，可以根据个人喜好进行选择。相当于一次提供三个主题，真是物超所值。
  • 菜单。配置菜单需要弄明白这几个对应关系：
    • item name: link（链接）
    • item name: 中文翻译（显示文字）
    • item name: icon name（显示图标）
  • 字体

另外可以通过一些第三方组件，为博客添加一些新的功能，比如：

  • 评论
  • 数据统计
  • 搜索
  • 分享

具体的配置方法可以参考主题的帮助文档。

Hexo提供的一些语法

  • 在要截取的位置插入<!--more-->，可以实现”查看更多“功能，否则首页上会显示全文。有不需要手动插入的方法，但是不建议使用。
